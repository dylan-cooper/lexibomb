    Lexibomb Server v0.0.1-dev LexibombServer.Board.Square

    Manages the low-level state data associated with an individual board square.

    A square can either be hidden or revealed, which indicates whether its state data should be available externally. Placing a tile on a square automatically sets it to the revealed state.
